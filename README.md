# Jenkins Configuration

## env
- Docker
- Docker Compose

## setup
```bash
sh setup.sh
docker compose build
docker compose up
```

## 初期設定
1. ブラウザで http://localhost:8080 にアクセス
2. 管理者アカウントの設定
3. 必要なプラグインの確認
